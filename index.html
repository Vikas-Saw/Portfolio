Projects I have worked on..

Heart Attack Prediction System.


I developed a machine learning-based Heart Attack Prediction System aimed at identifying potential heart attack risks using patient healthcare data. The project involved extensive data preprocessing, model training, and performance evaluation using Python, along with libraries such as Pandas, NumPy, and scikit-learn. The final model achieved an 85% accuracy rate on test data, offering a dependable tool for early risk assessment and supporting proactive healthcare decisions.

Graphical BMI Calculator and Health Tracker.


I created an interactive desktop application called the Graphical BMI Calculator and Health Tracker, designed to help users calculate their Body Mass Index (BMI) and monitor health trends over time. The application was built using Python’s Tkinter library for an intuitive graphical user interface and Matplotlib for real-time graph plotting. It enables users to input, track, and visualize their health parameters, promoting better awareness and management of personal fitness goals.
